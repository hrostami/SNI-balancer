# SNI-Balancer

یک بالانسر هوشمند برای کانفیگ‌های Xray در محیط‌های مبتنی بر SNI Spoofing

این پروژه به‌صورت خودکار کانفیگ‌های VLESS و Trojan را با تست واقعی سرعت دانلود و latency بررسی می‌کند، بر اساس عملکرد و پایداری به آن‌ها امتیاز می‌دهد و بهترین کانفیگ را به‌عنوان یک پراکسی SOCKS5 از طریق Xray اجرا نگه می‌دارد.

مناسب برای شبکه‌های دارای محدودیت و فیلترینگ که نیاز به SNI Spoofing دارند.

بدون نیاز به v2rayN یا کلاینت‌های GUI  
کاملاً خودکار

---

# امکانات

- دانلود و آپدیت خودکار Xray
- دانلود و مدیریت خودکار SNI-Spoofing
- پشتیبانی از نسخه Rust و Go
- تست واقعی سرعت از طریق پراکسی
- سیستم امتیازدهی مبتنی بر latency
- بررسی پایداری تاریخی کانفیگ‌ها
- Exponential Backoff برای کانفیگ‌های خراب
- Failover و Recovery خودکار
- داشبورد زنده با Rich TUI
- پشتیبانی از Subscription URL
- پشتیبانی از REALITY
- پشتیبانی از TLS
- چندسکویی:
  - لینوکس
  - ویندوز
  - macOS

# نحوه عملکرد

1. خواندن کانفیگ‌ها از `configs.txt`
2. ساخت نمونه‌های موقت Xray برای تست
3. انجام:
   - Health Check
   - تست Latency
   - تست واقعی سرعت دانلود
4. محاسبه امتیاز هر کانفیگ بر اساس:
   - سرعت
   - latency
   - پایداری
5. اجرای بهترین کانفیگ
6. تست مجدد در بازه‌های زمانی مشخص
7. سوییچ فقط در صورت بهتر بودن محسوس کانفیگ جدید
8. ذخیره تاریخچه تست‌ها
